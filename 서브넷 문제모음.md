문제 B) 서브넷 개수·호스트 요구 설계
네트워크 192.168.5.0/24 에서 최소 10개의 서브넷 각 서브넷당 최소 14대의 호스트를 수용하려 한다.

1. 필요한 서브넷 비트 수

2. 결정된 서브넷 마스크(/n)

3. 한 서브넷당 호스트 최대 수

4. 첫 5개 서브넷의 네트워크 주소와 브로드캐스트 주소

문제 C) IP 할당 확인
